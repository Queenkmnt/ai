การทาบรูปแบบง่ายๆ

 

  1. เนื้อหา
  2. การทาบรูปแบบง่ายๆ
สร้างข้อความแบบเส้นโค้ง การสร้างแสงกระจาย การสร้างไอคอน 3 มิติ การสร้างไอคอนโฟรเดอร์ การทาบรูปแบบง่ายๆ การสร้างดิสโก้บอล

การทาบรูปแบบง่ายๆ จาก illustrator ฉบับ Basic

  • 1. File -> New เลือกขนาดภาพ สัก 800 x 600 pt. ก็ได้
  •  2. File -> Place เลือกรูปที่จะเอามาทาบ
  • 3. เลือก ที่กล่องเลเยอร์ เพื่อสร้างเลเยอร์ใหม่ขึ้นมา เลือก New Layer
  • บนสุด ตั้งชื่อเลเยอร์นั้นให้เรียบร้อย ส่วนสีปกติมันก็เลือกให้อยู่แล้ว หรือถ้าอยากเปลี่ยนก็ได้ แต่มันไม่สำคัญอ่ะ แค่บอกว่าสีไหนอยู่เลเยอร์ไหน เวลาที่เราคลิกดู path เสร็จแล้วก็กดปุ่ม OK
  • 4. เริ่มทาบกันเลยนะ เราจะเริ่มจากคนซ้ายสุด
  • วิธีการทาบ มีดังนี้เลือก pen tool นะ กดเลือก Fill ให้อยู่ด้านหน้า stroke แล้วกด ไม่ใช้สี กดเลือก Stroke ให้อยู่หน้า Fill แล้วกด ไม่ใช้สี (คือ วาดแต่ path ไม่ต้องลงสีนะ) แล้วก็คลิ๊กๆ ไปตามแขนซ้าย แขนขวา หน้า ผม เสื้อ กางเกง รองเท้า สิ่งที่ต้องทำมีแค่นี้เองส่วนรายละเอียดเราค่อยมาลงทีหลัง ดูตัวอย่างจากรูปด้านล่างเลย
  • ตอนนี้ในเลเยอร์ที่ชื่อ 1 ของเราก็จะมีส่วนที่เป็น path 2 อัน อยู่ในเลเยอร์ 1 นะ จะสังเกตได้ว่า path แต่ละ path ที่เราสร้าง จะอยู่ในเลเยอร์ 1 และเป็นสีชมพูตามที่เราเลือกน่ะ เพื่อจะได้แยกออกว่าเรากำลังทำอยู่ในเลเยอร์ไหน เสร็จจากแขนแล้วก็ลองไปทำส่วนอื่นๆ ดูนะ ส่วนที่เราคิดว่ายากสุด ก็คงจะเป็นรองเท้า ถ้าทาบจะยากมาก เราก็เลยคิดเองเลย คิดว่ามันควรจะออกมาเป็นยังไง ตามรูปต้นแบบน่ะ อย่าลืมแยกองค์ประกอบของคนออกเป็นส่วนๆนะ ตามสีที่จะเติม จินตนาการเอาบ้างก็ได้ ไม่ต้องเอาเป๊ะๆ
  • วิธีทำให้ เส้น path โค้งได้รูป สามารถทำได้โดย เลือก Smooth Tool แล้วก็ลากไปตาม path ที่เราเลือก ก็จะเห็นว่า เส้นเรียบขึ้น เพราะถ้าไม่ทำให้ smooth มันก็จะเป็นรอยหยักๆ น่าเกลียด 
  • 5. ได้เวลาลงสีแล้ว เลือก path ทีละอันนะ ยกตัวอย่างแขนด้านซ้ายละกัน
  • เลือก path ที่เป็นแขนซ้าย โดย คลิ๊กให้ปุ่มเป็นสีดำหรื คลิ๊กส่วนที่คิดว่าเป็นเส้น pathแล้วเส้น path ก็จะปรากฏออกมาเอง
  • ลงสี มีวิธีอยู่หลายแบบ
  • 1. นำ Eyedropper มาดูดสีตามจริง โดยถ้าเวอร์ชั่น11 หรือ cs แค่เอาที่ดูดมาดูดตรงรูปได้เลย แต่ถ้าเวอร์ชั่น 10 ก็ต้องกด shift ไปด้วย 
  • 2. ดับเบิ้ลคลิ๊ก ตรง Fill แล้วเลือกสีเอา
  • 3. เลือกจาก กล่อง color เอาเม้าส์ไปจิ้มที่แถบสี ด้านล่างน่ะ 
  • เสร็จแล้วก็ไปลงสีส่วนอื่นๆ ให้หมดนะ 
  • TipS 
  • เวลาที่เราวาด path เช่นที่แขนนี้ เราอาจจะวาดเลยไปถึงแขนเสื้อก็ได้ แล้วพอลงสี ก็เอาเลเยอร์เสื้อ มาทับเลเยอร์แขน เท่านี้ก็จะได้path ที่ทับกันสนิท โดยจะเอาไปประยุกต์กับส่วนอื่นก็ได้ แค่ดูว่าเลเยอร์ไหนควรอยู่ก่อน เลเยอร์ไหนควรอยู่หลัง 
  • Outline 
  • ดูได้จาก การกด Ctrl+Y หรือ เลือกเมนู view ด้านบน แล้วเลือก Outline เป็นการดูเส้น path ทั้งหมดที่เราวาด
  • เลอร์เยอร์ทั้งหมด
  • สามารถตั้งชื่อ path หรือ เลเยอร์ โดยดับเบิ้ลคลิ๊กนะ แล้วก็ใส่ชื่อ
